# Stellaris 4.3.4 Domain Knowledge

> **Use when:** Verifying game mechanics, understanding empire design, or validating
> that code references real Stellaris 4.3.4 features.

## Ethics
Militarist, Pacifist, Xenophile, Xenophobe, Materialist, Spiritualist, Egalitarian, Authoritarian
(each can be Fanatic). Gestalt: Hive Mind, Machine Intelligence.

## Government Types
Imperial (ruler 80%), Democracy (~20% ruler), Oligarchy (~40% ruler), Dictatorial (70% ruler),
Hive Mind (single voice), Machine Intelligence (logic modules).

## Key Origins
Prosperous Unification, Void Dwellers, Shattered Ring, Life-Seeded, Necrophage, Scion,
Synthetic Fertility, Hegemon, Common Ground, Doomsday, Clone Army.

## Resources
Energy, Minerals, Food, Alloys, Consumer Goods, Influence, Unity.
Strategic: Volatile Motes, Exotic Gases, Rare Crystals, Zro, Dark Matter, Living Metal, Nanites.

## Ship Classes
Corvette (start), Destroyer (early), Cruiser (mid), Battleship (mid-late),
Titan (late), Colossus (perk), Juggernaut (late + perk).

## 4.3.4 Specifics
- Pop growth is S-curve with empire size
- Starbase cap matters for chokepoints
- Admin cap affects research/unity penalties
- Fleet comp counters are real (PD vs missiles)
- Megastructures require ascension perks
